## 2. Distinction from third-party projects

The Aura Sky / SKYGRID by Aura-Core™ project should not be confused with any outside project using names such as **Sky Protocol** or similar blockchain protocol branding.

Known distinction:

- Aura Sky / SKYGRID by Aura-Core™ is centered on backup-ready local infrastructure, pilot node participation, outage recovery, proof trails, and community-accessible resilience.
- Any third-party project called Sky Protocol or similar may have a different scope, ownership, history, technical stack, and commercial purpose.

Public materials should avoid using the generic phrase **Sky Protocol** as a standalone product name unless referring specifically to an unrelated third-party project for comparison or conflict review.

## 4. Project mission

Aura Sky / SKYGRID by Aura-Core™ is intended to support:

- Backup connectivity during ISP outages
- Resilience during grid and power instability
- Local node participation with consent-based onboarding
- Proof trails through GitHub, health JSON, and deployment records
- Pilot-stage node leasing exploration
- Safe mainstream Web3 / Layer 2 routing concepts
- Recovery paths across devices, browsers, and connectivity options

Potential connectivity and fallback concepts may include Ethernet, Wi-Fi, cellular, satellite, LoRa, MQTT, AC/DC-supported hardware, coaxial infrastructure, and other safe, auditable routes.
