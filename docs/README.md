## About Aura Core

Aura Core is a comprehensive wallet and payment infrastructure that combines:
- Multi-protocol wallet support (Meshtastic, Bluetooth, Coinbase)
- Lightning Network integration
- Biometric authentication
- Currency conversion and payment processing
- Real-time device synchronization
