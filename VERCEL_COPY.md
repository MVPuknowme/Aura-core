## Homepage section copy

### What SKYGRID does

SKYGRID helps teams see route health, test disruption scenarios, review network pressure, and make faster operational decisions without guessing. It brings together runtime health checks, advisory failover logic, Base utilization bands, Sun-Pay quote estimates, and Postman-ready validation routes in one clean interface.

### Why it matters

Modern systems depend on many fragile paths: cloud regions, APIs, payment rails, wallets, dashboards, validators, and local devices. SKYGRID gives those paths a visible command surface so operators can detect problems earlier, respond with evidence, and avoid risky blind automation.

### Built for proof-first operations

The current runtime is advisory-safe. Public routes do not move money, sign transactions, activate devices, or perform automatic OS-level switching. Every major action is designed to pass through validation, operator approval, and route proof before production activation.

## Feature cards

### Runtime health

Live health and route status endpoints make the platform inspectable from browsers, Postman, Vercel, and future dashboards.

### Dispatcher demo

A mobile-friendly advisory dispatcher demonstrates how SKYGRID can rank fallback paths during outages or congestion.

### Scenario testing

Operators can simulate carrier congestion, cloud slowdown, Base fee pressure, and other disruption patterns before real events occur.

### Highway API

A four-lane bridge map connects GitHub proof, Vercel runtime, Postman validation, AWS health mirrors, and Web3 status checks.

### Base rate bands

Adaptive utilization bands estimate green, yellow, and red pricing posture using network pressure signals.

### Sun-Pay quote logic

Quote-only payment estimates show settlement cost, utilization fee, and routing posture without moving funds or requiring private keys.

## Safety and trust copy

SKYGRID is designed to stay safe during public testing. The demo routes are advisory, transparent, and fail-closed. They do not perform automatic network switching, money movement, wallet signing, device entitlement activation, or emergency-service replacement.
